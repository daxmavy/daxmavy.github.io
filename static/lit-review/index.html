Normative Definitions of Political Bias in Large Language Models

What each study treats as the unbiased reference point — a general bias functional per family, with every paper instantiating its variables.

Compiled June 2026 · ~60 works verified against arXiv / ACL Anthology / publisher pages · companion to lit_review_normative_definitions.md

How to read this. Each family (A–H) states a general bias functional \(\mathrm{Bias}(M)\); the model is unbiased iff it vanishes. The table below each definition has one row per paper, instantiating the family's abstract symbols (\(x,\,s,\,c,\,D_\mathrm{ref},\dots\)) for that study. Notation is shared across families — see the key. Family M treats the cross-cutting question of controlling for content quality. NEW marks papers added to your Zotero import file.

Shared notation

\(M\)the model under evaluation
\(x,\;\;y=M(x)\)an input (prompt, test item, query, or document set) and the model's output
\(g\)a group of people — partisan, issue-position, or demographic
\(\phi(\cdot),\;\;\theta_M\)a feature/position map, and the model's resulting position in a value space
\(c\)the reference ("neutral") point
\(s(\cdot)\)a scalar score — reward, approval, agreement, similarity, or stance
\(Q(\cdot)\)a quality / helpfulness functional
\(D_M,\;\;D_{\mathrm{ref}}\)the model's output distribution and a reference distribution
\(d(\cdot,\cdot)\)a divergence or distance between distributions
\(\mathcal{V},\;\;\mathrm{cov}(y)\)the set of admissible perspectives, and those covered by output \(y\)
\(\sigma\)perceived slant (signed)
\(\Delta\)a difference, or a causal/treatment effect
\(\mathrm{Bias}(M)\)the family's bias functional; \(M\) is unbiased iff it vanishes (or \(\theta_M=c\), \(D_M=D_{\mathrm{ref}}\), …)

A

Instrument-centre norms

Unbiased = the political test's own midpoint. Administer a questionnaire, read off a position.

General definition
$$\theta_M=\tfrac{1}{T}\textstyle\sum_{t=1}^{T}\phi\bigl(M(x_t)\bigr),\qquad \mathrm{Bias}(M)=\lVert\theta_M-c\rVert,\qquad M\text{ unbiased}\iff\theta_M=c.$$

where \(x_t\) = test items, \(\phi\) = the test's scoring map into a value space, \(\theta_M\) = the model's position, \(c\) = the instrument's neutral point. Variant (Motoki): replace the fixed \(c\) with self-symmetry — regress the default answers on a partisan persona's, \(\theta^{\mathrm{def}}=\beta_0+\beta_1\theta^{\mathrm{persona}}+\varepsilon\), unbiased \(\iff\beta_1=0\).

B

Partisan symmetry — matched pairs, party-proximity & self-anchored

Unbiased = no systematic favouring of one side of the partisan dimension. Three operationalisations: B1 twin pairs differing only in direction; B2 per-statement alignment to party blocs (no twins); B3 the default output sitting halfway between the model's own most-partisan outputs.

General definition
$$\textbf{B1}\;\;\mathrm{Bias}(M)=\Delta=\mathbb{E}_{(x^{L},x^{R})\sim\mathcal{T}}\!\left[\,s\bigl(M,x^{L}\bigr)-s\bigl(M,x^{R}\bigr)\,\right],\qquad M\text{ unbiased}\iff\Delta=0.$$ $$\textbf{B2}\;\;a(M,p)=\frac{1}{|S|}\sum_{x\in S}\mathbb{1}\!\left\{\operatorname{stance}_M(x)=\operatorname{pos}_p(x)\right\},\qquad M\text{ unbiased}\iff a(M,\cdot)\ \text{balanced across opposing blocs.}$$ $$\textbf{B3}\;\;\mathrm{Bias}(M)=\bigl|\,d\bigl(y_0,y^{L}\bigr)-d\bigl(y_0,y^{R}\bigr)\,\bigr|,\qquad M\text{ unbiased}\iff y_0\ \text{equidistant from the model's own poles}\ y^{L},y^{R}.$$

B1 — matched pair: \(\mathcal{T}\) = twin pairs \((x^{L},x^{R})\) differing only in political direction, \(s\) = the score compared across the pair (Gubelmann variant: \(s=e\), an error indicator, so competence cannot be read as bias). B2 — party-proximity: \(S\) = a set of policy statements each scored individually, \(\operatorname{stance}_M(x)\in\{\text{agree, neutral, disagree}\}\), \(\operatorname{pos}_p(x)\) = party \(p\)'s official position, \(a(M,p)\) = alignment to party \(p\) (a voting-advice-application match); unbiased reference usually implicit (equidistance across the party space). B3 — self-anchored: \(y_0\) = the default output, \(y^{L},y^{R}\) = the model's own explicitly left/right-prompted outputs, \(d\) = a distance; the reference poles are generated by the model itself, so neutral ≠ an external standard but "halfway between what this model would say if pushed each way."

C

Even-handedness & calibration

Symmetric treatment across mirrored inputs. Two sub-norms: equal quality/helpfulness given a stance request (frontier labs), and calibration / statistical independence of generated ideology from innocuous attributes. Stance-taking on request is allowed; the bias is the asymmetry.

General definition
$$\text{(quality)}\;\;E(M)=\mathbb{E}_{x}\,\mathbb{1}\!\left\{\bigl|Q\bigl(M,x^{L}\bigr)-Q\bigl(M,x^{R}\bigr)\bigr|<\epsilon\right\},\qquad M\text{ unbiased}\iff E(M)=1.$$ $$\text{(calibration)}\;\;\Pr\bigl(y{=}1\mid\psi\cap o\bigr)=\Pr\bigl(y{=}1\mid\psi\bigr)\ \ \forall o,\qquad\text{target}\ \ \Pr(y{=}1)\to\tfrac12.$$

where \((x^{L},x^{R})\) = mirrored left/right framings of one request, \(Q\) = a task-conditional quality/helpfulness functional, \(\epsilon\) = a tolerance; for the calibration sub-norm \(y\) = an ideology-classifier label, \(o\) = an innocuous attribute (state, topic). Note: the calibration sub-norm (Liu) measures the slant of generated content, not its helpfulness — a distinct mechanism grouped here for its shared symmetry logic. Bang's anchor-equidistance and Vijay's pole-separability are not here — they are self-anchored symmetry (B3).

D

Distribution-matching norms

Unbiased = the output distribution equals a reference distribution. The family closest to the gatekeeping question in the DPhil — distinguished by which distribution is the reference.

General definition
$$\mathrm{Bias}(M)=d\!\left(D_M,\,D_{\mathrm{ref}}\right),\qquad M\text{ unbiased}\iff D_M=D_{\mathrm{ref}}.$$

where \(D_M\) = the model's output distribution (answers, mentions, recommended items, citations), \(D_{\mathrm{ref}}\) = the reference, \(d\) = a divergence. Sub-families differ only in \(D_{\mathrm{ref}}\):  D1 a surveyed population;  D2 real-world distributions (polls, results, salience);  D3 the input pool/source the system was given (gatekeeping fidelity).  D1* = representativeness of the feedback population, not the output; D3+ = D3-dominant but mixed with an absolute-balance norm.

E

Pluralism / diversity norms

Unbiased = adequate coverage of perspectives — within one response (Overton) or across the output distribution.

General definition
$$\text{OvertonScore}(M)=\mathbb{E}_{y\sim M}\!\left[\frac{|\mathrm{cov}(y)|}{|\mathcal{V}|}\right]\!,\;\;\text{ideal}\iff\mathrm{cov}(y)=\mathcal{V};\qquad\text{distributional:}\;\;P_M=P_G.$$

where \(\mathcal{V}\) = the set of admissible ("reasonable") perspectives on a question, \(\mathrm{cov}(y)\) = those represented in response \(y\), \(P_G\) = a population's perspective distribution. Diversity variants set the ideal as \(\mathrm{Div}(M)\ge\mathrm{Div}(\text{external baseline})\).

F

Perception / approval norms

Unbiased = humans — across opposing groups — judge it unbiased, or approve of it equally.

General definition
$$\sigma_g(M)=\mathbb{E}_{u\in g}\bigl[\sigma\bigl(M(x)\bigr)\bigr],\;\;M\text{ unbiased}\iff\sigma_g(M)=0\;\forall g;\qquad\text{(approval)}\;\;y^{*}_x=\arg\min_{y\in\mathcal{P}(x)}\bigl|s_1(x,y)-s_2(x,y)\bigr|.$$

where \(u\) = a user in group \(g\), \(\sigma\) = signed perceived slant, \(s_g\) = group \(g\)'s approval of output \(y\), \(\mathcal{P}(x)\) = the Pareto frontier of \((s_1,s_2)\). Stray's Maximum Equal Approval requires both balance and Pareto-optimality (refusal is balanced but off-frontier).

G

Downstream-effects norms

Unbiased = exposure produces no differential shift in opinions, votes, or decisions relative to a control.

General definition
$$\mathrm{Bias}(M)=\mathrm{ATE}=\mathbb{E}[\Delta\mid M]-\mathbb{E}[\Delta\mid\text{control}],\qquad M\text{ unbiased}\iff\mathrm{ATE}=0\;\text{and}\;\mathbb{E}[\Delta\mid M,g]\;\text{constant in }g.$$

where \(\Delta\) = a user's opinion / vote / decision shift, the treatment = exposure to model \(M\), the control = a neutral or status-quo comparison, \(g\) = partisan subgroup (no heterogeneity allowed).

H

Norm-sceptical & validity critiques (and the measures they motivate)

Mostly not a rival reference point — arguments that the measurement, or the idea of a fixed neutral point, fails first. Two entries (Faulborn, Wachter) go further and build their own bias metric in response — flagged as rival-measure, not pure critique.

General definition (the critique form)
$$\text{identifiable}\iff\mathrm{Var}_{\pi\in\Pi}\!\bigl(\theta_M^{\pi}\bigr)\to 0;\qquad\text{Törnberg:}\;\;\theta_M=f(a),\;\;\bigl|\partial\theta_M/\partial a\bigr|\;\text{large}.$$

where \(\Pi\) = a set of meaning-preserving perturbations (paraphrase, option order, forcing prompt), \(\theta_M^{\pi}\) = the position measured under \(\pi\), \(a\) = the inferred auditor/asker. A position is only a "bias" if stable; for some, no fixed point exists.

M

Mediating factors & quality control

The central methodological question: does any study separate political slant from confounds such as content quality?

The decomposition the literature mostly does not do
$$\underbrace{\mathrm{TE}}_{X\to Y}=\underbrace{\mathrm{NDE}}_{\text{direct }X\to Y}+\underbrace{\mathrm{NIE}}_{X\to Q\to Y}.$$

where \(X\) = the protected attribute (party / stance), \(Y\) = measured or perceived slant, \(Q\) = a quality mediator (truth, length, fluency, persuasiveness, relevance). By-design control fixes \(Q\) so \(\mathrm{NIE}=0\) by construction; by-statistics control estimates \(\mathrm{NDE}\) vs \(\mathrm{NIE}\) — almost absent. Where it is done, the partisan gap turns out largely \(\mathrm{NIE}\), or \(Q\) absorbs an apparent capability effect.

The gap, precisely. No LLM-political-bias study performs Imai/Pearl-style causal mediation of content quality; none uses IV, difference-in-differences, or propensity matching on quality covariates. The only formal mediation of a quality-type variable lives in the persuasion literature (Hackenburg). In the information-processing setting (summarisation, selection, RAG) the DPhil targets, no study conditions a perspective-level bias estimate on document quality — a perspective-extraction design that measures quality (or perplexity) and adjusts inclusion-bias for it would be the first of its kind.
