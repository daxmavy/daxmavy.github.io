Power & cost for a three-arm persuasion experiment¶

Against / neutral / for summaries — informed by Röttger et al.'s ai-distortion persuasion follow-up (Study 5). Every number below is computed from that public data; re-running this notebook shallow-clones the repo automatically.

The three questions this answers:

  1. How noisy is belief change, and how big is the effect we'd be chasing?
  2. Can we pick persuadable propositions ahead of time, using data Paul already collected?
  3. For a given assumed effect, how many participants and how many £ — and is it cheaper to ask each person 6 questions or 3?

1. The outcome, the effect, and the noise¶

Outcome is the oriented stance shift = direction × (attitude_post − attitude_pre) on a 0–100 scale, where direction points towards the position the paragraph argues. The static control (an irrelevant paragraph) is Paul's placebo arm.

Conservative noise. Rather than splitting the noise into reader/proposition/residual and using the small residual, we use the marginal within-condition SD — it lumps every variance component together and so over-states the noise that actually applies to a within-participant contrast (where reader variance cancels).

3. An ex-ante selection rule from the main study¶

Baseline reader uncertainty predicts persuadability — but it's measured inside the persuasion study, so you can't select on it without running it. Instead we use writer self-reports from the main study (collected for every proposition long before): how confident/knowledgeable writers were, and how far their own stance sat from the midpoint.

The outcome to predict is each proposition's mean persuadability in the follow-up.

Does it hold out of sample? Leave-one-proposition-out cross-validation, comparing a ridge model on the seven main-study features against the in-study uncertainty ceiling.

X = dat[['w_conf','w_know','w_import','w_stance_mean','w_stance_sd','w_extremity','w_self_change']].values
yv = dat.persuadability.values
internal = df.assign(e=(df.pa_pre-50).abs()).groupby('proposition_id').e.mean().reindex(dat.index).values

def loo_corr(Xmat):
    Xmat = Xmat.reshape(len(yv), -1); pred = np.zeros(len(yv))
    for tr, te in LeaveOneOut().split(Xmat):
        sc = StandardScaler().fit(Xmat[tr])
        mdl = RidgeCV(alphas=np.logspace(-2,3,20)).fit(sc.transform(Xmat[tr]), yv[tr])
        pred[te] = mdl.predict(sc.transform(Xmat[te]))
    return pearsonr(pred, yv)[0]

print(f'Main-study features (7, ridge):   corr(pred, actual) = {loo_corr(X):.3f}')
print(f'In-study uncertainty (ceiling):   corr(pred, actual) = {loo_corr(internal):.3f}')

Takeaway. Writer stance-extremity (tie-broken on low writer confidence) recovers ~90% of the achievable signal with zero new data. The realised lift is modest — roughly +15–35% effect size for the softest 10–20 propositions — because between-proposition variance is small to begin with.

4. Power and cost¶

Estimand: biased arm vs neutral arm (Δ = the assumed mean shift; we sweep Δ ∈ {1,2,5,10} because the neutral arm won't behave like Paul's irrelevant control). Manipulation is within-participant, so the contrast is paired and reader variance cancels. 90% power, two-sided 5%, σ from §1. Cost model is Paul's own: £12/rater-hour, 1.5 min/paragraph + 2.5 min overhead.

Design A = 6 questions (2 per arm); Design B = 3 questions (1 per arm).

At a realistic effect of Δ ≈ 3–4 (§1), £400 is roughly the floor for the biased-vs-neutral contrast on Design A, and £800 gives comfortable headroom (MDE ≈ 3.0). Both budgets detect the pro-vs-anti gap (≈ 2Δ) easily.

6. Recommendations¶

  • Design A (6 questions, 2 per arm) beats the 3-question design by ~20% at every effect size: the extra paragraphs halve the within-participant contrast variance and are cheap against the fixed 2.5-min overhead, with no fatigue penalty (§2).
  • Make the primary test pro-vs-anti (gap ≈ 2Δ if symmetric) rather than biased-vs-neutral — roughly 4× cheaper than the table above.
  • Select propositions by lowest writer stance-extremity (§3), softest ~12–15. Twelve-plus keeps the 6-question design free of within-person proposition repeats and gives enough proposition-clusters to generalise.
  • Plan around Δ ≈ 3–4, σ = 17. Strongly-stanced paragraphs in Paul's data moved beliefs ~3.4–4.5 pts; summaries may move them less, so treat Δ ≥ 5 as optimistic.

Caveats. Single contrast at α = 0.05 (testing both pro and anti vs neutral with Bonferroni adds ~20% to N); the paired model assumes within-person observations are independent given the participant (different propositions ⇒ reasonable); issue selection is inherently capped because the proposition ICC is ~0.02.
